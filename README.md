# Frontend - Baixinho (React + Vite)

## Rodar localmente
1. Instale dependências:
   ```
   cd frontend
   npm install
   ```
2. Desenvolvimento:
   ```
   npm run dev
   ```
3. Build para produção:
   ```
   npm run build
   ```

## Deploy no Netlify
1. Crie um novo site no Netlify e conecte ao repositório/zip.
2. Configure o build command: `npm run build` e o diretório `dist`.
3. O site será público em: https://baixinho.netlify.app (ou outro domínio Netlify escolhido).
